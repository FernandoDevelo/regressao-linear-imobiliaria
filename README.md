

# 🏠 Previsão de Preços de Imóveis com Regressão Linear

Projeto de Machine Learning desenvolvido com foco em regressão linear para previsão de preços de imóveis a partir de variáveis do conjunto de dados.

O objetivo deste projeto foi aplicar conceitos fundamentais de Ciência de Dados e Machine Learning, passando pelas etapas de análise exploratória, preparação dos dados, treinamento do modelo e avaliação dos resultados.

---

## 🚀 Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Statsmodels

---

## 📊 Etapas do projeto

Durante o desenvolvimento do projeto foram realizadas as seguintes etapas:

- Importação e tratamento dos dados
- Análise exploratória (EDA)
- Análise de correlação entre variáveis
- Separação entre treino e teste
- Construção do modelo de Regressão Linear
- Avaliação do desempenho do modelo
- Análise dos resíduos

---

## 🤖 Modelo utilizado

O modelo principal utilizado foi:

- Regressão Linear

Também foram utilizadas bibliotecas estatísticas para análise dos coeficientes e métricas do modelo.

---

## 📈 Resultados obtidos

O modelo apresentou o seguinte desempenho:

```python
R² = 0.738
```

Isso indica que o modelo conseguiu explicar aproximadamente **73,8% da variabilidade dos preços dos imóveis**.

A distribuição dos resíduos apresentou comportamento próximo de uma distribuição normal, indicando um ajuste satisfatório do modelo aos dados.

---

## 📉 Análise dos resíduos

A análise gráfica dos resíduos mostrou:

- resíduos concentrados próximos de zero
- distribuição aproximadamente normal
- presença de alguns outliers
- leve assimetria à esquerda

Esses resultados sugerem que o modelo possui boa capacidade preditiva, embora ainda exista espaço para melhorias futuras.
